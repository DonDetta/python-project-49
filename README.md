# Brain Games

Набор консольных мини-игр для тренировки мозга:

- проверка чётности числа (`brain-even`);
- калькулятор (`brain-calc`);
- поиск наибольшего общего делителя (`brain-gcd`);
- арифметическая прогрессия (`brain-progression`);
- простое ли число (`brain-prime`).

Проект выполнен в рамках обучающего курса Hexlet.

---

## Минимальные требования

- Python 3.10+ (рекомендуется та же версия, что в задании Hexlet)
- Установленный `pip` или менеджер зависимостей из проекта (`uv` / `poetry` — см. Makefile/pyproject)
- Unix-подобная система или WSL (Linux / macOS). На Windows запуск возможен через WSL.

---

## Установка

Склонируйте репозиторий и установите зависимости.

Пример (если используется `uv`, как в Makefile):

```bash
git clone https://github.com/DonDetta/python-project-49.git
cd python-project-49

# установка зависимостей
uv sync
```
